## Introduction
This is an audio resamplerate program based on [Secret Rabbit Code](http://www.mega-nerd.com/SRC/index.html) and [iniparser](http://ndevilla.free.fr/iniparser).

## Compile and Run
Compile with code :
```
make
```

You can modify the [**config.ini**](https://github.com/AnSwErYWJ/AudioResamplerate/blob/master/config.ini) to configure the parameters, input PCM or WAV audio file, and output PCM audio file.

Then,run your program with :
```
sh init.sh
```

## Environment
+ Linux
+ POSIX C
